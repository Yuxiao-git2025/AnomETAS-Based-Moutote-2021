# AnomETAS-Based-Moutote-2021
MATLAB code for identifying the **anomalous foreshock stages**, based on the ETAS model and its corresponding probability distribution <br>
You may follow the **source**: <br>
|| Moutote, L., Marsan, D., Lengliné, O., & Duputel, Z. (2021). Rare occurrences of non-cascading foreshock activity in Southern California. Geophysical Research Letters, 48, e2020GL091757. https://doi.org/10.1029/2020GL091757
